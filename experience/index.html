Additionally, I worked on three separate hobby projects under Knitlabs:

Job Match Tool

An automated script monitors YCombinator Hackernews job postings, matches them with resumes, and sends SMS alerts for potential job matches using libraries and APIs for data extraction and analysis. This project involves crawling and API fetches for Open Jobs, using Large Language Models (LLMs) to read resumes and evaluate them with respect to each available job, crafting SMS messages based on the evaluation, and automatically sending notifications for available jobs using Twilio. The GitHub Repository for this project is available at https://github.com/picklehari/resume_alerts/tree/main.

Practicaly

A framework to enable an end-user to upload a pdf, url or a video to develop lecture notes, rank topic importance and generate question paper. This project involves reusable pipelines that convert website URLs, PDFs and videos to text, using speech-to-text models to transcribe videos, VLM Models (Llava) to describe images in PDFs and combining the extracted text, and Beautiful Soup to fetch content from websites. The project also includes embedding models to convert extracted text to an embedding vector, clustering on embedding models to identify similar texts, and LLM Models (llama3) to generate lecture notes and identify topic importance, and use lecture notes and topic importance to generate a question paper covering Conceptual, Application and Reasoning types of questions.

Mitre Aligned Detections

A framework to enable end-user to identify security threats based within their organization based on MITRE framework. This project involves crawling and scrapping of detections, their techniques and tactics for various potential cyber threats, using LLM Agent to go through detection and produce Chain of Thought prompts to enable code generation, and prompt engineering based on derived logic to generate python code.
